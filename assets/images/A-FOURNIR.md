# Images à fournir

Le site est déjà câblé pour afficher ces images automatiquement dès qu'elles sont déposées dans ce dossier (`assets/images/`) avec exactement le nom indiqué. Tant qu'un fichier est absent, un bloc "placeholder" stylé s'affiche à sa place — le site reste donc présentable en attendant les vraies photos.

Formats recommandés : `.jpg` (photos) et `.png` (logo, fond transparent). Compresser les photos autour de 150–300 Ko pour un chargement rapide.

| Fichier attendu | Utilisé sur | Contenu | Dimensions conseillées |
|---|---|---|---|
| `logo.png` | Toutes les pages (en-tête + pied de page) | Logo de l'école, fond transparent | 200×200 px |
| `hero-batiments.jpg` | Accueil | Les bâtiments de l'école | 900×900 px (carré) |
| `hero-salle-classe.jpg` | Accueil | Une salle de classe | 700×700 px (carré) |
| `hero-environnement.jpg` | Accueil | Environnement / cour de l'école | 700×700 px (carré) |
| `organisation-discipline.jpg` | Accueil | Organisation et discipline (rang, uniforme, etc.) | 1000×750 px |
| `ecole-historique.jpg` | École | Photo illustrant l'histoire de l'école | 1000×750 px |
| `direction.jpg` | École | La direction de l'école | 1000×750 px |
| `programme-maternelle.jpg` | Programme | Activités de la section maternelle | 1000×750 px |
| `programme-primaire.jpg` | Programme | Activités de la section primaire | 1000×750 px |
| `programme-secondaire.jpg` | Programme | Activités de la section secondaire | 1000×750 px |
| `carnaval-metiers.jpg` | Activités | Carnaval des métiers | 1000×750 px |
| `activites-culturelles.jpg` | Activités | Chants, danses, événements culturels | 1000×750 px |
| `sorties-educatives.jpg` | Activités | Sortie éducative | 1000×750 px |
| `projets-eleves.jpg` | Activités | Projets réalisés par les élèves | 1000×750 px |

Pour remplacer une image : déposer le fichier dans ce dossier avec exactement le nom ci-dessus, rien d'autre à modifier dans le code.
